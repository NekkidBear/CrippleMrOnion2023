This is an open-source implementation of "Cripple Mr Onion" as described in "Witches Abroad" and other works by Sir Terry Pratchett. The rules were written by Andrew Millard and the modifiers were developed by Andrew and Terry Tao.

The software is released in compliance with GPL 3.0, MIT, and other similar licenses. No copyright infringement is intended or implied.

The rules are found at https://discworld.fandom.com/wiki/Cripple_Mr_Onion_Full_Rules
